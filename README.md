# notebooklm-hrtech-study-guide
Projeto desenvolvido como desafio da DIO utilizando o NotebookLM para organizar estudos sobre HR Tech, People Operations, Departamento Pessoal e Recursos Humanos. O repositório reúne a curadoria de fontes, experimentos de engenharia de prompts, aprendizados e um miniguia de estudos apoiado por Inteligência Artificial.
